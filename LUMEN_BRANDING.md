## Icon Specifications

### Sizes
```

256x256px - App icon (macOS retina)
128x128px - Dock icon
64x64px  - Menu bar
32x32px  - Toolbar
16x16px  - Small UI elements
```

### File Formats
- **PNG**: Transparent background, 24-bit color + alpha
- **SVG**: Vector, scalable, web use
- **ICNS**: macOS app bundle icon

### Icon Usage
‚úÖ DO:
- Use on dark or light backgrounds
- Maintain aspect ratio
- Keep clear space around icon
- Use gradient version for primary placement

‚ùå DON'T:
- Rotate or skew
- Change colors (except for specific themes)
- Add drop shadows (already built-in)
- Place on busy backgrounds
